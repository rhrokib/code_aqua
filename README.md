# code_aqua (codename)

## A simple webapp project for Software Engineering Lab

### To run the project

* first install the required packages/modules
  
```bash
pip install -r requirements.txt

```

* Then from the terminal, run

``` bash
pyhton ./runserver.sh

```

### Language/Framework used

* Django
* SQLite3
* Bootstrap
* Javascript
* JQuery
